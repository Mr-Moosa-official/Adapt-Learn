# **App Name**: AdaptLearn

## Core Features:

- Adjustable Text Sizes: Allow users to adjust the text size for optimal readability.
- High-Contrast Themes: Offer multiple high-contrast themes, including a dark mode, to accommodate various visual needs.
- AI-Generated Captions & Transcripts: Automatically generate captions and transcripts for all video and audio content using AI.
- Simplified Navigation: Implement a clear and intuitive navigation system, fully compatible with screen readers.
- User Preference Storage: Store user preferences (text size, theme, etc.) locally to provide a consistent experience across sessions.
- Content Playback: Users are able to play and pause content in any modality. All relevant actions such as starting/stopping video or audio playback have labels and appropriate aria attributes to make the content accessible.

## Style Guidelines:

- Primary color: Calming green (#8FBC8F) to evoke a sense of tranquility and learning.
- Background color: Soft, desaturated green (#E5F5E5) for a comfortable viewing experience.
- Accent color: Muted yellow (#BDB76B) to highlight important elements and calls to action, while maintaining overall accessibility standards.
- Body and headline font: 'PT Sans', a humanist sans-serif offering both a modern look and warmth.
- Use simple, clear icons that are easily recognizable and understandable, with descriptive alt text for screen readers.
- Maintain a clean and spacious layout, with consistent spacing and clear visual hierarchy to prevent cognitive overload.
- Use subtle animations for transitions and feedback, avoiding any flashing or strobing effects that could trigger visual sensitivities.
